# Next.js Resume — Ivelisse Feliciano Hernandez

A single-page resume web app built with Next.js (App Router) and styled with Tailwind CSS utility classes.

## How to Run Locally

1. Clone the repository:
   
   git clone https://github.com/Ivelisse-Feliciano/nextjs-resume.git
   
2. Navigate into the project folder:
   
   cd nextjs-resume
   
3. Install dependencies:
   
   npm install
   
4. Start the development server:
   
   npm run dev
   
5. Open your browser and go to http://localhost:3000

## Features Implemented

- [x] Resume content written as JSX in src/app/page.js
- [x] Real personal information — no placeholder text
- [x] Semantic HTML elements (<header>, <main>, <aside>, <section>, <article>, <footer>)
- [x] All styling done with Tailwind CSS utility classes via className
- [x] Logical heading hierarchy (<h1> for name, <h2> for section titles, <h3> for job titles)
- [x] Professional Experience section with all roles
- [x] Education section
- [x] Technical Skills section
- [x] Leadership & Business Skills section
- [x] Contact information with working links (email, LinkedIn, GitHub)
- [x] Clean, readable layout at standard desktop viewport
- [x] No errors in browser console
- [x] Dev server starts successfully with npm run dev

## Known Issues / Limitations

- Layout is optimized for desktop; mobile layout may need further refinement
- No dark mode support at this time
